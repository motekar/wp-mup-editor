== Description ==
MUP Editor is a WordPress code editor for Must-Use Plugins (`wp-content/mu-plugins`).

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/mup-editor` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Plugins->MU-Plugin Editor to open the editor
